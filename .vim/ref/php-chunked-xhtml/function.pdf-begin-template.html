PDF_begin_template

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_begin_template — テンプレート定義を開始する [古い関数]

説明

PDF_begin_template ( resource $pdfdoc , float $width , float $height ) : int

新規テンプレートの定義を開始します。

この関数は PDFlib バージョン 7 で廃止されました。 かわりに PDF_begin_template_ext() を使用してください。
