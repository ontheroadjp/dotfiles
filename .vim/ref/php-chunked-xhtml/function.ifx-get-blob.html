ifx_get_blob

(PHP 4, PHP 5 < 5.2.1)

ifx_get_blob — BLOB オブジェクトの内容を返す

説明

ifx_get_blob ( int $bid ) : string

BLOB オブジェクトの内容を返します。

パラメータ

bid

BLOB オブジェクト ID。

返り値

BLOB の内容を表す文字列、あるいはエラー時に FALSE を返します。
