MongoGridFS::get

(PECL mongo >=1.0.8)

MongoGridFS::get — ファイルをデータベースから取得する

説明

public MongoGridFS::get ( mixed $id ) : MongoGridFSFile

パラメータ

id

探したいファイルの _id。

返り値

見つかった場合はそのファイル、見つからない場合は NULL を返します。
