maxdb_stmt_close

maxdb_stmt::close

(PECL maxdb >= 1.0)

maxdb_stmt_close -- maxdb_stmt::close — プリペアドステートメントを閉じる

説明

手続き型

maxdb_stmt_close ( resource $stmt ) : bool

オブジェクト指向型

maxdb_stmt::close ( void ) : bool

プリペアドステートメントを閉じます。 maxdb_stmt_close() は、 stmt が指すステートメントハンドルを開放します。 現在のステートメントに処理中あるいはまだ読み込まれていない結果がある場合、 この関数はそれらをキャンセルし、次のクエリを実行できるようにします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
