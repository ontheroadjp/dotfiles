ifx_copy_blob

(PHP 4, PHP 5 < 5.2.1)

ifx_copy_blob — 指定した BLOB オブジェクトを二重化する

説明

ifx_copy_blob ( int $bid ) : int

指定した BLOB オブジェクトを二重化します。

パラメータ

bid

BLOB オブジェクトの ID。

返り値

新規の BLOB オブジェクト ID を返します。 エラーの場合に FALSE を返します。
