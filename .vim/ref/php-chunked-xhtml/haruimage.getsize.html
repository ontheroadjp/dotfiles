HaruImage::getSize

(PECL haru >= 0.0.1)

HaruImage::getSize — 画像の大きさを取得する

説明

HaruImage::getSize ( void ) : array

画像の大きさを取得します。

パラメータ

この関数にはパラメータはありません。

返り値

二つの要素 width および height を持つ配列を返します。 それぞれ、画像の対応する部分の大きさが含まれます。

エラー / 例外

エラー時に HaruException をスローします。
