stats_rand_gen_beta

(PECL stats >= 1.0.0)

stats_rand_gen_beta — ベータ分布から無作為な値を生成する

説明

stats_rand_gen_beta ( float $a , float $b ) : float

パラメータ A および B で表されるベータ分布から、無作為な値を返します。 ベータ分布の確率密度は、0 < x < 1 に対して x^(a-1) * (1-x)^(b-1) / B(a,b) となります。R. C. H. Cheng の手法を用います。

パラメータ

a

形状母数

b

形状母数

返り値

無作為な値
