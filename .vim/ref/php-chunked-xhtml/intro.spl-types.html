はじめに

この拡張モジュールを使用すると、PHP をより強力な型付けの言語として使用することができ、 スカラー型のタイプヒンティングの代替手段として使用できます。 integer や float、bool、enum などのさまざまな型処理用のクラスを提供します。

警告

この拡張モジュールは、 実験的 なものです。この拡張モジュールの動作・ 関数名・その他ドキュメントに書かれている事項は、予告なく、将来的な PHP のリリースにおいて変更される可能性があります。 このモジュールは自己責任で使用してください。
