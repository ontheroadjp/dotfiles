Mongo::__construct

(PECL mongo >=0.9.0)

Mongo::__construct — コンストラクタ

説明

public Mongo::__construct ([ string $server [, array $options ]] )

このメソッドは、MongoClient のコンストラクタを上書きして 確認付き書き込みを無効にします。

パラメータについての説明は MongoClient::__construct() を参照ください。

返り値

警告

このクラスのインスタンスを作ると E_DEPRECATED が発生し、確認付き書き込みが無効になります。

かわりに MongoClient を使いましょう。
