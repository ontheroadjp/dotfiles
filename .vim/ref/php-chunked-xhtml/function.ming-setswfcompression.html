ming_setswfcompression

(PHP 5.2.1-5.3.0, PHP 7, PECL ming SVN)

ming_setswfcompression — SWF 出力の圧縮を設定する

説明

ming_setswfcompression ( int $level ) : void

SWF 出力の圧縮レベルを設定する。

パラメータ

level

新しい圧縮レベル。1 から 9 までの値でなければなりません。

返り値

値を返しません。
