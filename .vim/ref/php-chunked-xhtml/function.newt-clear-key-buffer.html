newt_clear_key_buffer

(PECL newt >= 0.1)

newt_clear_key_buffer — 追加の入力を待たずに、端末の入力バッファの内容をクリアする

説明

newt_clear_key_buffer ( void ) : void

追加の入力を待たずに、端末の入力バッファの内容をクリアします。

返り値

値を返しません。
