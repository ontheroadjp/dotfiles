Counter::getMeta

Counter::getMeta — カウンタのメタ情報を返す

説明

Counter::getMeta ( integer $attribute ) : mixed

Counter::getMeta() は、カウンタのメタ情報を返します。

パラメータ

attribute
取得したいメタ情報。

返り値

Counter::getMeta() は、 指定されたメタ情報に応じてさまざまな型の値を返します。
