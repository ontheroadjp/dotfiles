stats_absolute_deviation

(PECL stats >= 1.0.0)

stats_absolute_deviation — 値の配列の絶対偏差を返す

説明

stats_absolute_deviation ( array $a ) : float

a に含まれる値の絶対偏差を返します。

パラメータ

a

入力配列

返り値

a に含まれる値の絶対偏差。 a が空の場合、または、配列ではない場合は FALSE を返します。
