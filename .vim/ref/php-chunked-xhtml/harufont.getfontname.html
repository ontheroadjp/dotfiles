HaruFont::getFontName

(PECL haru >= 0.0.1)

HaruFont::getFontName — フォント名を取得する

説明

HaruFont::getFontName ( void ) : string

フォントの名前を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

フォントの名前を返します。

エラー / 例外

エラー時に HaruException をスローします。
