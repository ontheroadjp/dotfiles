dbplus_unselect

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_unselect — リレーションから制約を削除する

説明

dbplus_unselect ( resource $relation ) : int

dbplus_unselect() をコールすると、 dbplus_find() により relation に設定された制約が削除されます。

パラメータ

relation

dbplus_open() でオープンしたリレーション。

返り値
