DOMNodeList クラス

(PHP 5, PHP 7)

クラス概要

DOMNodeList implements Traversable {
/* プロパティ */
readonly public int $length ;
/* メソッド */
public count ( void ) : int
DOMNodelist::item ( int $index ) : DOMElement
}

プロパティ

length

リスト内のノードの数。有効な子ノードのインデックスの範囲は 0 以上 length - 1 以下
