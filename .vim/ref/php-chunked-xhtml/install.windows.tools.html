Windows 上での PHP インストーラ

PHP インストールツール

IIS を使っている場合に PHP 本体やいくつかの PHP アプリケーションをインストールしようとしたときに いちばん簡単な方法は、Microsoft の Web Platform Installer (WebPI) を使うことです。

XAMPP や WampServer や BitNami などを使えば、Windows 上で Apache を使う場合の PHP アプリケーションをセットアップできます。
