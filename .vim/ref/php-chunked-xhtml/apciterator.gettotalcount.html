APCIterator::getTotalCount

(PECL apc >= 3.1.1)

APCIterator::getTotalCount — 総数を取得する

説明

public APCIterator::getTotalCount ( void ) : int

総数を取得します。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

この関数にはパラメータはありません。

返り値

総数を返します。
