はじめに

警告

この機能は PHP 7.0.0 で 削除 されました。

この機能の代替として、これらが使えます。

  • PDO_SQLSRV (Windows のみ)
  • PDO_ODBC
  • SQLSRV (Windows のみ)
  • Unified ODBC API

以下の関数によりMS SQL Server データベースにアクセス可能となります。

この拡張モジュールは、Windows 版では PHP 5.3 以降使うことができません。

SQLSRV という別の MS SQL 接続用拡張モジュールが、Microsoft から » http://msdn.microsoft.com/en-us/sqlserver/ff657782.aspx で公開されています。
