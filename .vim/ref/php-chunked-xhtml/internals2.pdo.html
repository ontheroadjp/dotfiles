この How-To の目的は、PDO 用レイヤーとのインターフェイスとなる データベースドライバを書くために必要な基礎知識を身につけることです。 この API はまだ発展途上のものであり、変更される可能性があることに注意しましょう。 このドキュメントは、PDO のバージョン 0.3 をもとにしています。 学習曲線はかなりの急勾配です。まず最初の段階を身につけるのにかなりの時間を要するでしょう。
