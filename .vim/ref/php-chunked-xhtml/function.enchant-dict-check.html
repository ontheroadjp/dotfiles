enchant_dict_check

(PHP 5 >= 5.3.0, PHP 7, PECL enchant >= 0.1.0 )

enchant_dict_check — 単語のスペルが正しいかどうかを調べる

説明

enchant_dict_check ( resource $dict , string $word ) : bool

単語のスペルが正しい場合に TRUE、 そうでない場合に FALSE を返します。

パラメータ

dict

辞書リソース。

word

調べる単語。

返り値

単語のスペルが正しい場合に TRUE、 そうでない場合に FALSE を返します。
