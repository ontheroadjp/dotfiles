stats_dens_pmf_poisson

(PECL stats >= 1.0.0)

stats_dens_pmf_poisson — ポアソン分布の確率質量関数

説明

stats_dens_pmf_poisson ( float $x , float $lb ) : float

母数 lb のポアソン分布にしたがう確率変数の x における確率質量を返します。

パラメータ

x

確率変数の値

lb

母数

返り値

x における確率質量。失敗時は FALSE を返します。
