KTaglib_Tag::getComment

(0.0.1)

KTaglib_Tag::getComment — ID3 タグからコメントを返す

説明

public KTaglib_Tag::getComment ( void ) : string

ID3 タグのコメントを返します。 このメソッドは、ID3v1 および ID3v2 タグで実装されています。

返り値

コメント文字列を返します。
