ReflectionParameter::export

(PHP 5, PHP 7)

ReflectionParameter::export — エクスポートする

説明

public static ReflectionParameter::export ( string $function , string $parameter [, bool $return ] ) : string

エクスポートします。

警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

パラメータ

function

関数名。

parameter

パラメータ名。

return

TRUE に設定すると、エクスポートを発行するのではなく返します。 FALSE (デフォルト) の場合はその逆です。

返り値

エクスポート結果を返します。
