HaruPage::getCurrentPos

(PECL haru >= 0.0.1)

HaruPage::getCurrentPos — パスの描画用の現在の位置を取得する

説明

HaruPage::getCurrentPos ( void ) : array

パスの描画用に使用する現在位置を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

パスの描画用に使用する現在位置を、二つの要素 "x" と "y" からなる配列で返します。

エラー / 例外

エラー時に HaruException をスローします。
