px_delete

(PECL paradox >= 1.0.0)

px_delete — paradox データベースのリソースを削除する

説明

px_delete ( resource $pxdoc ) : bool

paradox ファイルのリソースを削除し、メモリを開放します。

パラメータ

pxdoc

px_new() が返す paradox データベースのリソース ID。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
