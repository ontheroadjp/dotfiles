yp_order

(PHP 4, PHP 5 < 5.1.0)

yp_order — マップの呼出番号を返す

説明

yp_order ( string $domain , string $map ) : int

マップの呼出番号を返します。

パラメータ

domain

map

返り値

マップの呼出番号、あるいはエラー時に FALSE を返します。

例

例1 NIS 呼出番号の例

<?php
    $number = yp_order($domain, $mapname);
    echo "このマップの呼出番号は、" . $number;
?>
