XMLWriter::fullEndElement

xmlwriter_full_end_element

(PHP 5 >= 5.2.0, PHP 7, PECL xmlwriter >= 2.0.4)

XMLWriter::fullEndElement -- xmlwriter_full_end_element — 現在の要素を終了する

説明

オブジェクト指向型

XMLWriter::fullEndElement ( void ) : bool

手続き型

xmlwriter_full_end_element ( resource $xmlwriter ) : bool

現在の xml 要素を終了します。要素が空であっても終了タグを書き込みます。

パラメータ

xmlwriter

手続き型のコールでのみ使用します。 変更される XMLWriter resource です。 このリソースは、xmlwriter_open_uri() あるいは xmlwriter_open_memory() のコールによって取得したものです。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
