udm_hash32

(PHP 4 >= 4.3.3, PHP 5 < 5.1.0, PECL mnogosearch >= 1.0.0)

udm_hash32 — 指定した文字列の Hash32 チェックサムを返す

説明

udm_hash32 ( resource $agent , string $str ) : int

udm_hash32() は、文字列 str を受け取り、その 32 ビットハッシュ値を返します。

パラメータ

agent

事前の >udm_alloc_agent() のコールにより返されたエージェントへのリンク。

str

入力文字列。

返り値

32 ビットのハッシュ値を返します。
