SWFSprite クラス

(PHP 5 < 5.3.0, PECL ming SVN)

はじめに

SWFSprite は "ムービークリップ" ともいいます。これにより、作成したオブジェクトを 時間軸にそって動かすことができるようになります。 したがって、スプライトはムービーのメソッドの大半を保持しています。

クラス概要

SWFSprite {
/* メソッド */
add ( object $object ) : void
__construct ( void )
labelFrame ( string $label ) : void
nextFrame ( void ) : void
remove ( object $object ) : void
setFrames ( int $number ) : void
startSound ( SWFSound $sount ) : SWFSoundInstance
stopSound ( SWFSound $sount ) : void
}
