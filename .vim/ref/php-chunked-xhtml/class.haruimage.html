HaruImage クラス

(PECL haru >= 0.0.1)

はじめに

Haru PDF 画像クラスです。

クラス概要

HaruImage {
/* メソッド */

getColorSpace ( void ) : string
getHeight ( void ) : int
getSize ( void ) : array
getWidth ( void ) : int

}
