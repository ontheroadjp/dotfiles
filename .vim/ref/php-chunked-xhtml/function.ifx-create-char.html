ifx_create_char

(PHP 4, PHP 5 < 5.2.1)

ifx_create_char — 文字オブジェクトを作成する

説明

ifx_create_char ( string $param ) : int

文字オブジェクトを作成します。

パラメータ

param

内容となる文字。

返り値

新しい文字オブジェクトの ID、あるいはエラー時に FALSE を返します。
