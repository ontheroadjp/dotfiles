readline_write_history

(PHP 4, PHP 5, PHP 7)

readline_write_history — ヒストリを書きこむ

説明

readline_write_history ([ string $filename ] ) : bool

この関数はコマンドヒストリをファイルに書き込みます。

パラメータ

filename

保存するファイルへのパス。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
