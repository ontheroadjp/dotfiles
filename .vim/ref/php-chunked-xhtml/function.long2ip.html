long2ip

(PHP 4, PHP 5, PHP 7)

long2ip — 長整数評言のインターネットアドレスを (IPv4) インターネット標準ドット表記に変換する

説明

long2ip ( int $proper_address ) : string

関数long2ip() は、長整数表現のアドレスからドット表記 (例:aaa.bbb.ccc.ddd)のインターネットアドレスを生成します。

パラメータ

proper_address

長整数で表した、正しい形式のアドレス。

返り値

インターネットの IP アドレスを表す文字列を返します。
