udm_free_res

(PHP 4 >= 4.0.5, PHP 5 < 5.1.0, PECL mnogosearch >= 1.0.0)

udm_free_res — mnoGoSearch 結果を開放する

説明

udm_free_res ( resource $res ) : bool

結果用に確保されたメモリを開放します。

パラメータ

res

udm_find() から返された結果 ID へのリンク。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
