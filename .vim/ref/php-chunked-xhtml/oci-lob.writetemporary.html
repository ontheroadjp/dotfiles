OCI-Lob::writeTemporary

(PHP 5, PHP 7, PECL OCI8 >= 1.1.0)

OCI-Lob::writeTemporary — 一時的なラージオブジェクトを書き込む

説明

OCI-Lob::writeTemporary ( string $data [, int $lob_type = OCI_TEMP_CLOB ] ) : bool

一時的なラージオブジェクトを生成し、 data を書き込みます。

オブジェクトの使用後、OCI-Lob::close を使用するべきです。

パラメータ

data

書き込むデータ。

lob_type

以下のいずれかです。

  • OCI_TEMP_BLOB が一時的な BLOB を生成するために使われます
  • OCI_TEMP_CLOB が一時的な CLOB を生成するために使われます

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
