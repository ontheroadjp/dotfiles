stats_dens_uniform

(PECL stats >= 1.0.0)

stats_dens_uniform — 一様分布の確率密度関数

説明

stats_dens_uniform ( float $x , float $a , float $b ) : float

下限 a, 上限 b の一様分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

a

下限

b

上限

返り値

x における確率密度。失敗時は FALSE を返します。
