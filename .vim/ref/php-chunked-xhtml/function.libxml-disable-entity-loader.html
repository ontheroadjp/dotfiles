libxml_disable_entity_loader

(PHP 5 >= 5.2.11, PHP 7)

libxml_disable_entity_loader — 外部エンティティの読み込み機能を無効にする

説明

libxml_disable_entity_loader ([ bool $disable = TRUE ] ) : bool

外部エンティティ読み込み機能の有効/無効を切り替えます。

パラメータ

disable

libxml を使用する拡張モジュール (DOM、XMLWriter および XMLReader) で、外部エンティティの読み込み機能を 無効 (TRUE) あるいは有効 (FALSE) にします。

返り値

変更前の値を返します。
