px_numfields

(PECL paradox >= 1.0.0)

px_numfields — データベース内のフィールドの数を返す

説明

px_numfields ( resource $pxdoc ) : int

データベースファイル内のフィールドの数を返します。

パラメータ

pxdoc

px_new() が返す paradox データベースのリソース ID。

返り値

データベースファイル内のフィールドの数を返します。この関数の返す値は、 px_get_info() が返す配列における要素 numfields の値と等しくなります。
