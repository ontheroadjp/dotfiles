QuickHashIntSet::getSize

(PECL quickhash >= Unknown)

QuickHashIntSet::getSize — セットの要素数を返す

説明

publicQuickHashIntSet::getSize ( void ) : int

セットの要素数を返します。

パラメータ

key

エントリのキー。

返り値

セットの要素数を返します。

例

例1 QuickHashIntSet::getSize() の例

<?php
$set = new QuickHashIntSet( 8 );
var_dump( $set->add( 2 ) );
var_dump( $set->add( 3 ) );
var_dump( $set->getSize() );
?>

上の例の出力は、 たとえば以下のようになります。

bool(true)
bool(true)
int(2)
