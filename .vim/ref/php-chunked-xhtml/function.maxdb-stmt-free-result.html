maxdb_stmt_free_result

maxdb_stmt::free_result

(PECL maxdb >= 1.0)

maxdb_stmt_free_result -- maxdb_stmt::free_result — 指定したステートメントハンドルの結果を保存しているメモリを開放する

説明

手続き型

maxdb_stmt_free_result ( resource $stmt ) : void

オブジェクト指向型

maxdb_stmt::free_result ( void ) : void

maxdb_stmt_free_result() 関数は、 stmt パラメータで指定した結果メモリ (これは maxdb_stmt_store_result() によって確保されたものです) を開放します。

返り値

この関数は、なにも値を返しません。
