Counter::bumpValue

Counter::bumpValue — カウンタの現在値を変更する

説明

Counter::bumpValue ( integer $offset )

Counter::bumpValue() は、カウンタの現在値を更新します。

パラメータ

offset
カウンタの値を変更する量。負の値を指定することも可能です。
