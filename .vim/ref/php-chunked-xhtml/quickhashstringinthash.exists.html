QuickHashStringIntHash::exists — キーがハッシュに存在するかどうかを調べる

説明

public QuickHashStringIntHash::exists ( string $key ) : bool

指定したキーのエントリがハッシュに存在するかどうかを調べます。

パラメータ

key

ハッシュ内に存在するかどうかを調べたいキー。

返り値

エントリが存在する場合に TRUE、存在しない場合に FALSE を返します。
