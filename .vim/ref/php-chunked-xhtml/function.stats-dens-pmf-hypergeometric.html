stats_dens_pmf_hypergeometric

(PECL stats >= 1.0.0)

stats_dens_pmf_hypergeometric — 超幾何分布の確率質量関数

説明

stats_dens_pmf_hypergeometric ( float $n1 , float $n2 , float $N1 , float $N2 ) : float

失敗回数 n2, 成功サンプル数 N1, 失敗サンプル数 N2 の超幾何分布にしたがう確率変数の n1 における確率質量を返します。

パラメータ

n1

成功回数

n2

失敗回数

N1

成功サンプル数

N2

失敗サンプル数

返り値

n1 における確率質量。失敗時は FALSE を返します。
