session_pgsql_reset

(PECL session_pgsql SVN)

session_pgsql_reset — セッションデータベースサーバーとの接続をリセットする

説明

session_pgsql_reset ( void ) : bool

セッションデータベースサーバーとの接続をリセットします。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
