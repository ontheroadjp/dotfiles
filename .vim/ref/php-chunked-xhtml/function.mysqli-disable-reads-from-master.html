mysqli::disable_reads_from_master

mysqli_disable_reads_from_master

(PHP 5 < 5.3.0)

mysqli::disable_reads_from_master -- mysqli_disable_reads_from_master — マスタからの読み込みを無効にする

説明

オブジェクト指向型

mysqli::disable_reads_from_master ( void ) : void

手続き型

mysqli_disable_reads_from_master ( mysqli $link ) : bool
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

警告

この関数は 非推奨 であり、PHP 5.3.0 で 削除 されます。
