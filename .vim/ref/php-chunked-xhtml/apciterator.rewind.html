APCIterator::rewind

(PECL apc >= 3.1.1)

APCIterator::rewind — イテレータを巻き戻す

説明

public APCIterator::rewind ( void ) : void

イテレータを最初の要素に巻き戻します。

パラメータ

この関数にはパラメータはありません。

返り値

値を返しません。
