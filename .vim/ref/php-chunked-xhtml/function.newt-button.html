newt_button

(PECL newt >= 0.1)

newt_button — 新しいボタンを作成する

説明

newt_button ( int $left , int $top , string $text ) : resource

新しいボタンを作成します。

パラメータ

left

ボタンの X 座標。

top

ボタンの Y 座標。

text

ボタンに表示するテキスト。

返り値

作成したボタンコンポーネントへのリソースリンク、あるいはエラー時に FALSE を返します。

例

例1 newt_button() の例

<?php

$form = newt_form();

$ok_button = newt_button(5, 12, "Run Tool");
    
newt_form_add_component($form, $ok_button);

?>
