stats_dens_pmf_negative_binomial

(PECL stats >= 1.0.0)

stats_dens_pmf_negative_binomial — 負の二項分布の確率密度関数

説明

stats_dens_pmf_negative_binomial ( float $x , float $n , float $pi ) : float

成功回数 n, 成功確率 pi の負の二項分布にしたがう確率変数の x における確率密度を返します。

パラメータ

x

確率変数の値

n

成功回数

pi

成功確率

返り値

x における確率密度。失敗時は FALSE を返します。
