新しいクラス

PHP 5.4.0 で新しく加わったクラスは以下の通りです。

  • ReflectionZendExtension

  • JsonSerializable インターフェイス

Session:

  • SessionHandler クラス
  • SessionHandlerInterface

Snmp:

  • SNMP

Intl:

  • Transliterator
  • Spoofchecker
