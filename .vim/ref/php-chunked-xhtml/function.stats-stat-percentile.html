stats_stat_percentile

(PECL stats >= 1.0.0)

stats_stat_percentile — パーセンタイル値を返す

説明

stats_stat_percentile ( array $arr , float $perc ) : float

配列 arr に含まれる値の perc パーセンタイル値を返します。

パラメータ

arr

入力配列

perc

パーセンタイル

返り値

入力配列のパーセンタイル値を返します。
