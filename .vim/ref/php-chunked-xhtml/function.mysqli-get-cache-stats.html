mysqli_get_cache_stats

(PHP 5 >= 5.3.0 and < 5.4.0)

mysqli_get_cache_stats — クライアントの Zval キャッシュの統計情報を返す

警告

この関数は PHP 5.4.0 で 削除されました。

説明

mysqli_get_cache_stats ( void ) : array

空の配列を返します。 mysqlnd でのみ使用可能です。

パラメータ

返り値

成功した場合に空の配列、それ以外の場合に FALSE を返します。

変更履歴

バージョン 説明
5.4.0 mysqli_get_cache_stats() が削除されました。
5.3.0 mysqli_get_cache_stats() がスタブとして追加されました。
