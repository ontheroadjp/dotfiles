HaruFont::getEncodingName

(PECL haru >= 0.0.1)

HaruFont::getEncodingName — エンコーディング名を取得する

説明

HaruFont::getEncodingName ( void ) : string

フォントエンコーディングの名前を取得します。

パラメータ

この関数にはパラメータはありません。

返り値

フォントエンコーディングの名前を返します。

エラー / 例外

エラー時に HaruException をスローします。
