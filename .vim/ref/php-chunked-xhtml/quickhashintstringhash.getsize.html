QuickHashIntStringHash::getSize

(PECL quickhash >= Unknown)

QuickHashIntStringHash::getSize — ハッシュの要素数を返す

説明

public QuickHashIntStringHash::getSize ( void ) : int

ハッシュの要素数を返します。

返り値

ハッシュの要素数を返します。

例

例1 QuickHashIntStringHash::getSize() の例

<?php
$hash = new QuickHashIntStringHash( 8 );
var_dump( $hash->add( 2, "two" ) );
var_dump( $hash->add( 3, 5 ) );
var_dump( $hash->getSize() );
?>

上の例の出力は、 たとえば以下のようになります。

bool(true)
bool(true)
int(2)
