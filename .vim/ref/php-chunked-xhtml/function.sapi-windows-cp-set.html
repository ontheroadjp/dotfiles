sapi_windows_cp_set

(PHP 7 >= 7.1.0)

sapi_windows_cp_set — Set process codepage

説明

sapi_windows_cp_set ( int $cp ) : bool

Set the codepage of the current process.

パラメータ

cp

A codepage identifier.

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。
