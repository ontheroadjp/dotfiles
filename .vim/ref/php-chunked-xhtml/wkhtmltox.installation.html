インストール手順

この拡張機能のソースコードと Windows 用バイナリは » github でホストされています。

ソースコードの取得と拡張機能のビルド： 

git clone https://github.com/krakjoe/wkhtmltox
cd wkhtmltox
phpize
./configure --with-wkhtmltox=/path/to/wkhtmltox/installation
make
sudo make install

更新の取得と拡張機能の再ビルド： 

cd wkhtmltox
phpize --clean
git pull origin master
phpize
./configure --with-wkhtmltox=/path/to/wkhtmltox/installation
make
sudo make install
