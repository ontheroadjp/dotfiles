PDF_add_thumbnail

(PHP 4 >= 4.0.5, PECL pdflib >= 1.0.0)

PDF_add_thumbnail — 現在のページにサムネイルを追加する

説明

PDF_add_thumbnail ( resource $pdfdoc , int $image ) : bool

現在のページにサムネイルとして既存のイメージを追加します。 成功した場合に TRUE を、失敗した場合に FALSE を返します。
