stats_stat_paired_t

(PECL stats >= 1.0.0)

stats_stat_paired_t — 対応のある二群の t 検定値を返す

説明

stats_stat_paired_t ( array $arr1 , array $arr2 ) : float

対応のある二群 arr1 と arr2 に対する t 検定値を返します。

パラメータ

arr1

ひとつめのサンプル

arr2

ふたつめのサンプル

返り値

t 検定値。失敗時は FALSE を返します。
