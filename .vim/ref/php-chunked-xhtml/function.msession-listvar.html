msession_listvar

(PHP 4 >= 4.2.0, PHP 5 < 5.1.3)

msession_listvar — セッションの一覧を変数を使用して取得する

説明

msession_listvar ( string $name ) : array

共通の属性を有するセッションを検索する際に使用します。

パラメータ

name

検索する名前。

返り値

name という名前の変数を有する全ての セッションを値とする連想配列を返します。
