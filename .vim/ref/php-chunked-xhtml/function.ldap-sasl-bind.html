ldap_sasl_bind

(PHP 5, PHP 7)

ldap_sasl_bind — SASL を使用して LDAP ディレクトリにバインドする

説明

ldap_sasl_bind ( resource $link [, string $binddn = NULL [, string $password = NULL [, string $sasl_mech = NULL [, string $sasl_realm = NULL [, string $sasl_authc_id = NULL [, string $sasl_authz_id = NULL [, string $props = NULL ]]]]]]] ) : bool
警告

この関数は、 現在のところ詳細な情報はありません。引数のリストのみが 記述されています。

返り値

成功した場合に TRUE を、失敗した場合に FALSE を返します。

注意

注意: 前提条件
ldap_sasl_bind() は SASL サポート（sasl.h）を必要とします。PHP の configure 時に --with-ldap-sasl が指定されている ことを確認してください。さもないとこの関数は未定義となります。
