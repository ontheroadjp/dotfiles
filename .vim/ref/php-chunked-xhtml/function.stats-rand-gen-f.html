stats_rand_gen_f

(PECL stats >= 1.0.0)

stats_rand_gen_f — F 分布から無作為な値を生成する

説明

stats_rand_gen_f ( float $dfn , float $dfd ) : float

分子の自由度が "dfn"、分母の自由度が "dfd" の F (分散比) 分布から、無作為な値を返します。カイ二乗の変量の比を直接生成します。

パラメータ

dfn

分子の自由度

dfd

分母の自由度

返り値

無作為な値
